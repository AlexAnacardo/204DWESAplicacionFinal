\doxysubsection{Detailed Description}
Clase para contener informacion de una foto

Esta clase representa una foto de la NASA y sus detalles asociados, tales como el título, la URL de la imagen y la explicación. Permite establecer y obtener estos valores mediante métodos getter y setter.
